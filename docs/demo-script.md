# 2-Minute Demo Script

This follows the public challenge recommendation of roughly 1 minute live demo, 30 seconds architecture, and 30 seconds vision.

## 0:00 to 0:15

“StoryCast turns one plain-English idea into a narrated short film. It writes the story, generates storyboard frames, synthesizes voiceover, animates each frame into a clip, and cuts the whole film automatically.”

## 0:15 to 1:00

1. Open the StoryCast web UI.
2. Enter a prompt like `the death of a star`.
3. Submit the job.
4. Show the live status feed as the pipeline moves through Seed 2.0, Seedream 5.0, Seed Speech, and Seedance 2.0.
5. Open the final video and play the assembled result.

## 1:00 to 1:30

“The system is scene-structured. Seed 2.0 outputs a strict JSON blueprint, each scene gets a keyframe from Seedream 5.0, narration from Seed Speech, and image-to-video animation from Seedance 2.0. ffmpeg then muxes and concatenates the clips into a final 60-second film.”

## 1:30 to 2:00

“After the hackathon, StoryCast becomes an always-on explainer engine for education, science communication, and brand storytelling. The same agent architecture can generate multi-format outputs: shorts, classroom modules, narrated social clips, and interactive topic journeys.”

## Suggested Topics For The Demo

- `the death of a star`
- `how coral reefs talk to each other`
- `what happens inside a black hole`
- `the final minute of a honeybee’s mission`
- `how memories form in the human brain`
